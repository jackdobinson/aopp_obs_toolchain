# Deconvolution Tool Chain Tutorials #

Each sub-folder in this directory holds a tutorial on a specific part of the deconvolution tool chain.

.					This folder (Root).
↳ data/ 			Holds example data files.
	↳ cache/		Holds any cached files used when running tutorials.
↳ papers/ 			Holds a collection of relevant papers
↳ scripts/ 			Holds all scripts needed by examples
↳ tex/				Contains latex files for ./tutorial.pdf
↳ tutorial_*_XXX/ 	Contains a tutorial example
	↳ output/		Contains the outputs of a tutorial example, usually with example data.
	↳ run_XXX.py	A script to show how a task is accomplished.
