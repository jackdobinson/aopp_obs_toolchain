\doxysubsection{Detailed Description}
\begin{DoxyVerb}    Class defines a gaussian where the shape and type are fixed at instantiaion, but the mean, standard deviation, and offset are calculated
    by the instance. This is useful for putting into an optimising function as the underlying array storage is only created once.\end{DoxyVerb}
