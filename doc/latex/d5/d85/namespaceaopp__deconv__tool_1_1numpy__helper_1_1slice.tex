        if a.shape=(3,5,7,9) and group=(1,3), then on iteration the indices
        `idx` select a slice from `a` such that a[idx].shape=(5,9).
    squeeze

        or should they remain separate (need a.ndim - len(group) loops).\end{DoxyVerb}
