Detailed Description

    Lets the user set and retrieve referenced variables like a pointer does.
    Not sure exactly how to emulate pointers to normal variables yet.

    USAGE
    a_ptr = Pointer(7)
    b_ptr = Pointer(8)

    a_ptr.ref = b_ptr.ref
    print(a_ptr) # '8'

    b_ptr.val = 10
    print(a_ptr) # '10'
